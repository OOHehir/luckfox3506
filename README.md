# Luckfox3506 KiCad Project

This repository contains the KiCad design files for breadboard friendly adapter board for the LuckFox Core3506 microcontroller module.

## Overview

Hardware design project utilizing KiCad v9 for PCB layout and schematic capture.

## References

- [LuckFox Core3506 Module](https://www.luckfox.com/Core3506)
- [Wiki Page](https://wiki.luckfox.com/Core3506)

## Contents

- Schematic files
- PCB layout files
- Component libraries
- Design documentation

## Requirements

- KiCad 9.0 or later

## Getting Started

1. Clone this repository
2. Open the project files in KiCad
3. Review the schematic and PCB layout
